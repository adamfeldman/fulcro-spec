== Development

To run cljs tests:

     lein figwheel

To run clj tests:

     lein test-refresh

=== CI Testing

To run the CLJ and CLJS tests on the CI server, it must have chrome, node, and npm installed. Then
you can simply use the Makefile:

    make tests
